# Newspaper Portal

Newspaper Portal is a modern, dynamic web application that provides users with the latest news articles from various categories. Built with **React.js, Tailwind CSS**, and integrated with external **News APIs**, it offers a clean and responsive interface for browsing news.  

---

## 🚀 Features

### User Features
- **Responsive Design:** Works perfectly on mobile, tablet, and desktop devices.  
- **News Categories:** Browse news by categories such as Technology, Sports, Entertainment, Politics, etc.  
- **Search Functionality:** Search for news articles by keywords.  
- **Real-Time Updates:** Fetches the latest news from external APIs.  
- **Bookmark Articles (Optional):** Save favorite articles locally using browser storage (if implemented).  

**Frontend:** React.js, Tailwind CSS, HTML, JavaScript  
- **API:** External News API (e.g., [NewsAPI.org](https://newsapi.org))  
- **Deployment:** Vercel / Netlify
- 🌐 Live Demo : https://eact-dragon-news-auth-main.vercel.app/
